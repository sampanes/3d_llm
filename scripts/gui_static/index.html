What this does

          
This GUI is a friendly shell over the repository scripts. It lists model folders and STL files, builds commands like .venv\Scripts\python -m scripts.validate_stl ..., runs them as background jobs, and shows the output.

        

        

          
What it does not do

          
It does not replace model.py, edit_stl, build_model, validate_stl, or mesh_preview. Those scripts remain the business logic and source of truth.

        

        

          
Workflow

          
Select a model or mesh, choose an action, adjust flags, read the command, then run it. The GUI does not hide the script layer.

        

        

          
Viewer

          
Use the mouse to rotate and zoom the STL preview. Snap buttons reset the camera to repeatable views.

        

        

          
Commands

          
The command panel shows exactly what will run. The rows beneath it explain positional arguments and flags.

        

        

          
Outputs

          
Generated files stay under the project tree. Use Refresh after external file changes.
